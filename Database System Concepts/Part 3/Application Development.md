
### 9.1 - Application Programs and User Interfaces

pg 403

- _Application Program_ - Provides a user interface at the front end and interfaces with a database at the backend. Takes input from the users and generate output.


### 9.2 - Web Fundamentals

- _Uniform Resource Locator_ - A globally unique name for each document that can be accessed on the web
- _Web Server_ - A program running on the server machine that accepts requests from a web browser and sends back results in the form of HTML documents
- _Common Gateway Interface_ - Defines how the webserver communicates with the database, through ODBC, JDBC, or other protocols, in order to get data.
- HTTP is connectionless, that is, most computers have limits on the number of sites on the web open connections they can accommodate, and if a large number of sites on the webpage open connections to a single server, this limit would be exceeded, denying service to further users. With a connectionless protocol, the connection can be broken as soon as a request is satisfied

### 9.3 - Servlets

- _servlet_ - Defines an application programming interface for communication between the web/application server and the application program

### 9.5 - Web Services

pg 426

- _web service_ - An application component that can be invoked over the web and functions, in effect, like an application programming interface
- _REST (Representation State Transfer)_ - Web service function calls are executed by a standard HTTP request to a URL at an application server, with parameters sent as standard HTTP request parameters
